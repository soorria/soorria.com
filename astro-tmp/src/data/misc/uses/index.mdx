## Software

### Editor & Teminal

My main editor is [Visual Studio Code](https://code.visualstudio.com/) with
these main extensions:

- [Dracula Official](https://marketplace.visualstudio.com/items?itemName=dracula-theme.theme-dracula)
  (theme)
- [Vim](https://marketplace.visualstudio.com/items?itemName=vscodevim.vim) -
  should be self-expanatory
- [Auto Rename Tag](https://marketplace.visualstudio.com/items?itemName=formulahendry.auto-rename-tag) -
  makes renaming / changing HTML and JSX tags much easier

When I'm stuck in a terminal, or just hacking on some scripts I use
[neovim](https://neovim.io/) with
[this](https://github.com/soorria/wsl-dotfiles/blob/master/config/nvim/init.vim)
`init.vim`,

I use WSL through [Windows Terminal](https://github.com/Microsoft/Terminal) and
use Bash as my shell. For my shell prompt I use [Starship](https://starship.rs/)
with
[this](https://github.com/soorria/wsl-dotfiles/blob/master/config/starship.toml)
config.

For my terminal and editor, the font I use is the
[JetBrains Mono](https://www.jetbrains.com/lp/mono/) nerd font. All my
current-ish dotfiles are [here](https://github.com/soorria/wsl-dotfiles), and
the scripts I use are [here](https://github.com/soorria/scripts).

### Browser

I use Brave with these extensions:

- [Dark Reader](https://darkreader.org/) - makes "light" sites by heathens more
  palatable for my eyes
- [uBlock Origin](https://ublockorigin.com/) - sometimes Brave's adblocker isn't
  enough
- [VisBug](https://chrome.google.com/webstore/detail/visbug/cdockenadnadldjbbgcallicgledbeoc/) -
  nice for measuring things when ~stealing~ getting inspiration from other
  websites
- [OctoLinker](https://github.com/OctoLinker/OctoLinker) - converts imports into
  links in GitHub
- [Refined Github](https://github.com/refined-github/refined-github) - makes
  GitHub a little nicer
