Make some keys on a type optional. Opposite of
[PartiallyRequired](/snippets/partially-required).

```typescript nojs
type PartiallyOptional<T, K extends keyof T> = Omit<T, K> & Partial<Pick<T, K>>
```

## Usage

```typescript nojs
type User = {
  name: string
  email: string
}

// Type where `email` is optional, but `name` is still required
type UserWithEmailOptional = PartiallyOptional<User, 'email'>
```
